Pace documentation
==================

Pace is an implementation of the FV3GFS / SHiELD atmospheric model developed by NOAA/GFDL using the GT4Py domain-specific language in Python.
The model can be run on a laptop using Python-based backend or on thousands of heterogeneous compute nodes of a large supercomputer.
